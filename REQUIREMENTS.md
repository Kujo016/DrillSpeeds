# Requirements

## Operating System
- **Windows**: The build scripts (`win\build_drill.bat`) and execution scripts (`run_drill_speeds.bat`) are designed for Windows environments.

## Build System Dependencies
- **[minimum_project](https://github.com/Kujo016/minimum_project)**: The `DrillSpeeds` repository uses the `minimum_project` repository for its core builder system. You will need to refer to its scripts and tools to compile the project.

## Compiler
- A compatible C++ compiler accessible via the Windows command line (e.g., MSVC used by `minimum_project`'s builder tool).
